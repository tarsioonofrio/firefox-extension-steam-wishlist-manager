# Docs Index

Documentation is organized by area:

- `docs/dev/`
  - `DEV_WORKFLOWS.md`: local development workflows and command aliases.

- `docs/ops/`
  - `OPERATIONS.md`: operational runbook for recovery and incident handling.
  - `FIREFOX_DEVTOOLS_MCP_RUNBOOK.md`: Firefox DevTools MCP setup and troubleshooting.

- `docs/architecture/`
  - `ARCHITECTURE.md`: module boundaries, orchestration model, and design rules.
